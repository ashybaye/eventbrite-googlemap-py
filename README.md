# eventbrite-googlemap-py

Install steps:

1. $ python3
2. git clone <name_of_repo>
3. cd eventbrite-googlemap-py
4. $ python3
5. $ python3 -m venv venv
6. $ virtualenv venv
7. $ venv\Scripts\activate
  (venv) $ _
8. (venv) $ pip install flask
9. (venv) $ export FLASK_APP=eventbrite-googlemap.py  
10. (venv) $ flask run
